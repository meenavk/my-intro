# 🌐 Personal Portfolio Website

🚀 A modern, modular, animation-driven personal portfolio built using **HTML, CSS, and Vanilla JavaScript**.

This project is designed with a **component-based architecture (no frameworks)** where UI sections, styles, animations, and logic are fully separated for scalability, performance, and maintainability.

It represents my work in **engineering, automation systems, PLC/HMI development, and frontend web development**.
