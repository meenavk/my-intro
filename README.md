# 🌐 Personal Portfolio Website

A modular, animation-driven personal portfolio website built using **HTML, CSS, and Vanilla JavaScript**.

This project is designed with a **component-based architecture without frameworks**, where UI sections, styles, animations, and logic are separated for scalability and maintainability.

It showcases my engineering background, automation experience, technical skills, and software development projects.

## 🎯 About This Project

This portfolio was created to:

- Present my engineering and automation experience
- Showcase PLC, HMI, and software projects
- Demonstrate frontend architecture using Vanilla JavaScript
- Build a scalable UI system without frameworks
- Serve as a professional digital resume

---

## ✨ Features

- Modular section-based architecture
- JavaScript-based dynamic content loading
- Separate CSS files for each UI section
- Smooth animations (typing, fade-in, landing effects)
- Project gallery with image modal
- Navbar scroll tracking & highlighting
- Role-based experience structure
- Fully responsive design
- Lightweight (no frameworks)
